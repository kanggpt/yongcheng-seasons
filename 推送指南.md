## 🎯 操作步骤（在手机上完成）

### 第 1 步：在 GitHub 创建仓库

1. **打开 GitHub**
   - 手机浏览器访问：https://github.com
   - 登录您的账号（kanggbpt）

2. **创建新仓库**
   - 点击右上角 **+** → **New repository**
   - **Repository name**: `yongcheng-seasons`
   - **描述**: 邕城四季公寓官网 - 24 节气文化主题公寓
   - **Public**: 选中（公开）
   - 点击 **Create repository**

---

### 第 2 步：上传文件

**方法 A：直接上传（最简单）**

1. 创建仓库后，点击 **uploading an existing file**
2. 把 `index.html` 和 `README.md` 拖进去
3. 点击 **Commit changes**

**方法 B：使用 GitHub App**

1. 下载 GitHub 手机 App
2. 打开您的仓库
3. 点击 **Add file** → **Upload files**
4. 选择文件上传

---

### 第 3 步：启用 GitHub Pages

1. 打开您的仓库：https://github.com/kanggbpt/yongcheng-seasons
2. 点击 **Settings**（设置）
3. 左侧菜单找到 **Pages**
4. **Source** 选择 `main` 分支
5. 点击 **Save**

---

### 第 4 步：访问网站

等待 1-2 分钟后，在手机浏览器访问：

```
https://kanggbpt.github.io/yongcheng-seasons/
```

**就可以看到了！** 🎉
