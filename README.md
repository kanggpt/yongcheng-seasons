## 🚀 部署说明

### 本地预览

直接打开 `index.html` 文件即可预览网站。

### 部署到 GitHub Pages

1. **创建 GitHub 仓库**
   ```bash
   # 在 GitHub 上创建新仓库，例如：yongcheng-seasons
   ```

2. **推送代码**
   ```bash
   git remote add origin https://github.com/你的用户名/yongcheng-seasons.git
   git branch -M main
   git push -u origin main
   ```

3. **启用 GitHub Pages**
   - 打开 GitHub 仓库
   - Settings → Pages
   - Source 选择 `main` 分支
   - Save

4. **访问网站**
   ```
   https://你的用户名.github.io/yongcheng-seasons/
   ```

### 自定义域名（可选）

1. 购买域名（如：yongcheng-apartment.com）
2. 在 GitHub Pages 设置中添加自定义域名
3. 配置 DNS 解析
